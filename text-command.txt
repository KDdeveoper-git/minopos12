Laravel 
    -php artisan serve          : ເປັນຄຳສັ່ງ Run project
    -php atisan key:generate    : ເປັນຄໍາສັ່ງສ້າງ key app
    -composer install           : ຄຳສັ່ງຕິດຕັ້ງ Ribrary ຂອງ Laravel

VueJS

    -npm run device             : ເປັນຄຳສັ່ງ compile VueJS

Git

    - git commit -m "ຄຳອະິບາຍ"   : ຄຳສັ່ງ commmit
    - git push                  : ຄຳສັ່ງ ອັບໂຫຼດໂປລເຈັກ ຂຶ້ນ github
    - git pull                   : ຄຳສັ່ງ ອັບເດດ Source code ຈາກ    github
